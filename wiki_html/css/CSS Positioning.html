Box model

我们可以把HTML中的元素 element (<p>, < div>, <h1> ... )当作盒子.

box可以分成四层，从内到外依次为：

  • content: 实际内容
  • padding: 在content外一层,但还是在element中
  • border: 介于margin与padding
  • margin: box外面 
margin可以是 *负数*,
margin仅仅表示box外边的空间大小。
正的表示空间变大，负的表示空间缩小。

padding却不可以<0.

floating images

float

x动起来，可以试试 float:[left|right]

float可以让元素移动到左或右，然后让其他元素围绕它，就像报纸一样。

可以通过div并设置float来移动某些元素，如 

div {
  float: right;
}

如果多个元素使用float怎么办？

比如两幅图都是float:right,那么第一副图会移到右上角， 而第二副图移到第一个左边,以此类推，直到空间不够，再在下面一层

  • 参考练习codecademy
  • w3cschool css float

我们可以通过设置margin大小来分开两个图 

*当然margin可以是负数*

clear

在float元素后面的元素，会包围在它们左右。如果我们不想这样， 可以使用clear:[left|right|both]。那么接下来它们会重新换行。

不会对之后的float元素产生影响 

div#nav{
  clear: left;
}

参考css_float

position

首先了解下block与inline的区别......

相对位移是相对的。不像float，其他元素还是认为 它在原来的位置上，即使它动了。

要使用，先要 

position:relative;

之后设置top,right,bottom,left
