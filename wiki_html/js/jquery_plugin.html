自动补全

.autocomplete( json )

例子 

var availableTags = [
    "ActionScript",
    "AppleScript",
    "Asp",
];
$(function() {
    $('#tags').autocomplete({source:availableTags});
});

tabs

.tabs()

需要按照一定的格式进行编辑。参考如下: 

    <div id="tabs">
        <ul>
            <li><a href="#tabs-1">Nunc tincidunt</a></li>
            <li><a href="#tabs-2">Proin dolor</a></li>
            <li><a href="#tabs-3">cross fire</a></li>
        </ul>
        <div id="tabs-1">
            <p>Proin elit arcu, rutrum commodo......</p>
        </div>
        <div id="tabs-2">
            <p>Morbi tincidunt, dui sit amet facilisis feugiat, odi</p>
        </div>
        <div id="tabs-3">
            <p>fire in the hole.抄袭cf</p>
        </div>
    </div>

标签的名字在ul里定义，内容在对应的div#tabs-n中。
