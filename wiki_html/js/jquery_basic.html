selector

jQuery()函数是整个jQuery库的重点，常用快捷方式$()代替。

这个函数返回jQuery对象,有很多方法可调用.

标签名

// select every div on the page
$('div')

ID

// select a jQuery object with the element that has the id 
$element = $('#elementId')

只有一个

class

// select all elements of a certain class
$('.my-class')

组合

descendent selector 

$("div .thingy")

child selector 

$(".thingy > a")

........

.last( )

遍历DOM

.children( 空|selector ) 返回所有直接子元素

  return $that.children();
}

.find( selector ) 返回所有子孙，并满足条件 

$spanan = $('#i').find("span");

.parent( ) 父元素

.parents( ) 祖先

.closest( ) 寻找最近的某个祖先

.siblings( )

变更元素

jQuery中很多方法都可以操纵元素。很多既是getter，又是setter。 取决与是否传递了参数。

html

.html() 返还所选html内容 

$myEle.html(); // 获取
$myEle.html("<b>Hi</b>"); // 设置

.width( )

.height( )

css

.css( ) 

// 只有一个参数就是getter，两个就是setter
$el.css("color");
$el.css("color", "black");

不要过多使用

  1. 选择元素多速度就慢
  2. 代码复杂

可以使用归类方法.

归类

.addClass( class ) 添加传递的类名

.removeClass( ) 如果没有参数，移除所有类

addClass, removeClass可以对selector使用

.hasClass()

.toggleClass( ) 没有某类的话添加，否则移除
