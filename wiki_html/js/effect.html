特效

    $('div.bounce').click(function(){
        $(this).effect("bounce",{time:3},300);
    });

动画

.animate( properties, [duration], [erasing], [complete])

properties: a map of CSS properties that the animate will move towards 

  width = $('#block').width();
  height = $('#block').height();
  //jib
  size = parseInt($('#block').css('font-size'), 10);
  //your code here
  $('#block').click(function(){
      width += 50;
      height += 50;
      size += 2;

      $(this).animate({
        "height": width+"px", "width":width+"px",
        "font":size+"px"}, 1500);
  });

show hide

.show/hide(duration, [,complete])

animate, fade and hide 区别

$(document).ready(function(){
//your code here
	$(".items").mouseover(function() {
    	$("a", this).show( , function(){ $(this).animate({width:100}});    
	});
	$(".items").mouseout(function() {
    	$("a", this).animate({width:0}, function(){ $(this).hide();});
	});
});

selector 可以加第二个参数，作为上下文

$(this).find('a')
