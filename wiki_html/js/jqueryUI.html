$('.selector').draggable("option", "addClasses")

draggable

.draggable()

可以任意拖拽，相应地还有droppable函数，可以添加落下相应动作。

option

  • containment: 可以指定selector, element, or "parent"等来进行移动区域限制。
  • cursor: 指定cursor样式以及方位等: "crosshair"...
  • helper: "clone",拖拽时clone自己
  • revert: 复位, 值为true那么一定复位，"invalid"则在只有droppable才不复位
  • connectToSortable: 与sortable对象链接，可以转换位子，配合helper："clone"使用，因为可能它的位子会被挤走。。。。。。

droppable

用于防置拖拽对象

option

  • accept: selector|function, 表示可以防止的对象
  • activeClass: 当可放置的对象被拖拽时，把这个类添加到对象
  • hoverClass: 当可放置的对象被拖到对象上面时候，添加这个类

example

  $( "#cart ol" ).droppable({
    activeClass: "ui-state-default",
    hoverClass: "ui-state-hover",
    accept: ":not(.ui-sortable-helper)",
    drop: function( event, ui ) {
      $( this ).find( ".placeholder" ).remove();
      $( "<li></li>" ).text( ui.draggable.text() ).appendTo( this );
    }
  })

sortable

jQuery UI可以干很多很牛的事情，比如sortable()函数。

它可以自动让 list 里面的元素可以被拖动，而且可以重新安排 顺序。

option

  • placeholder: 类名，用以代替移动时原来位置的空白。
