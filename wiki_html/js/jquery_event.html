click

很简单

on

$('document').ready( function(){} );

//等于
$('document').on('ready', function(){} );

delegate与on的区别见下

多事件

传递一个{ eventName: function }对象即可 

$('#sample').on({
  mouseenter: function(e){ alert('enter');},
  mouseleave: function(e){ alert('leave');}
});

事件目标

事件函数的参数event，通过event.target获取目标 

$('button').click(function(e){
  $target = $(e.target);
  alert(  $target );
});

mouse,键盘，hover，focus

自己查文档

.hover( funcIn, funcOut)

.focus() <--> .blur()

高级，自定义事件

trigger

    //自定义事件，trigger触发
    $('#myElementId').on('las',function(e){
        $(this).addClass('green');
    });
    
    $('#myElementId').trigger('las');

delegate

delegate与on的区别是delegate会对document ready之后创建的元素 也附加handler。 

$(document).ready(function(){
    $('body p').delegate('click', function(){
        $(this).after('No click me!');
    });
    $('body').append('<p>fk me</p>');
});
