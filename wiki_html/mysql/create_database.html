创建数据库和用户

mysql> create database dbname;

GRANT

GRANT privileges [columns]
ON item
TO user_name [INDENTIFIED BY 'passwd']
[REQUIRE ssl_options]
[WITH [GRANT OPTION | limit_options]
  • privileges: 权限名，如select, insert .....
  • columns: 对一列或逗号分开的多列指定权限
  • item: 数据库或表名。常见形式：dbname.*, 全局权限*.*
  • username: 数据库用户名，可在后面包含一个主机名， jack@somewhere.com
  • GRANT OPTION: 被授权的人可以授权

REVOKE

REVOKE privileges [( columns )]
ON item
FROM user_name

如果有WITH GRANT OPTION 

REVOKE ALL PRIVILEGES, GRANT OPTION
FROM user_name

索引

当某些数据需要经常查询，我们需要添加索引来改进性能。 

CREATE [UNIQUE | FULLTEXT] INDEX index_name
ON table_name (index_column_name [(length)] [ASC|DESC], ...);
